# lung-cancer-subphenotyping
Lung Cancer Sub-phenotyping using MCA and Consensus Clustering.
MSc Health Data Science dissertation project (University of St Andrews, 2023). Unsupervised consensus clustering pipeline applied to multimodal clinical and genomic data from 490 lung cancer patients to identify clinically distinct subphenotypes. Methods: Multiple Correspondence Analysis, k-means, agglomerative hierarchical clustering, Gaussian Mixture Models.
